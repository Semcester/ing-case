# **Project Name**

This project is a simple employee management application for ING case study created by Semih Senan. Follow the steps below to install dependencies, start the application, and run tests.

---

## **Setup and Run**

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start the Application**:
   ```bash
   npm start
   ```
   This command will start the development server and open the application in your browser.

3. **Run Tests**:
   ```bash
   npm test
   ```
   This command will run all unit tests and display the results in the console.
